Grid Overview

Stylus Boilerplate ships with two types of grid systems:

  1. Traditional float-style grid on fixed columns
  2. Flexible, flex-box/table-based grid with unlimited columns

Flexible Grid

Because flexbox isn't yet supported in older browsers, and is less performant than table layouts, I adapted a grid based off of mdo'stable grid.

To use the grid system, simply wrap your elements in a.rowclass, followed by any column elements within a.columnclass.
